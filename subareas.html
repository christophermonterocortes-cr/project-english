Subareas of the Specialty

Six formative pillars that structure the profile of the Web Development student.

01

Information Technologies

IT fundamentals, networks, operating systems and hardware. The technical base upon which the entire specialty is built.

02

Entrepreneurship & Innovation

Agile methodologies, project management and creative thinking to transform ideas into real tech products.

03

Web Programming

HTML, CSS, JavaScript, Python, PHP and C#. Frontend, Backend and Full-Stack development with a practical approach to real projects.

04

Software Design

Systems architecture, UI/UX principles and database modeling to build robust and scalable applications.

05

IT Support

Preventive and corrective maintenance, LAN/WLAN network configuration and professional technical incident response.

06

English Oriented to Web Dev

Technical vocabulary, reading official documentation and professional communication in English for the web development environment.
